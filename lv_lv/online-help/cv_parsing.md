# CV parsēšana

CV parsēšana ekonomē kandidātu laiku un palielina pieteikumu konversijas rādītāju, jo kandidātiem nevajag manuāli ievadīt personisko informāciju pieteikuma veidlapā. Izmantojot CV parsēšanu, kandidāti var augšupielādēt CV failu no sava Dropbox, Google Drive konta vai datora un tas tiks pārvērsts strukturētos datos, kas aizpilda pieteikuma veidlapu. Pēc strukturētās informācijas iesniegšanas lietotāji to var ērti meklēt.

## To jautājumu saraksts, kas var tikt aizpildīti

Ja kandidātu CV failā ir attiecīgā informācija un jautājums ir iekļauts pieteikuma veidlapā vai karjeras centra CV veidlapā, var tikt aizpildīta šāda informācija:

-   Personas dati
    -   Uzruna
    -   Vārds
    -   Otrs vārds
    -   Uzvārds
    -   Dzimums
    -   Dzimšanas datums
    -   E-pasts
    -   Atkārtot e-pastu
    -   Mājas telefons
    -   Mobilais telefons
    -   Adrese
    -   Pasta indekss
    -   Pilsēta
    -   Ģimenes stāvoklis
    -   Augšupielādēt attēlu
-   Darba pieredze
    -   Periods
    -   Darba devējs
    -   Amats
    -   Amata apraksts
    -   Pašreizējais amats
-   Izglītība
    -   Periods
    -   Mācību iestāde
    -   Eksāmenu rezultāts, tituls
    -   Pašreizējā izglītība
-   Valodas
    -   Valoda
    -   Runātprasme
    -   Lasītprasme
    -   Rakstītprasme
-   Sociālā tīkla profila URL
    -   LinkedIn
    -   Xing
    -   Facebook

## Atbalstītie dokumentu formāti un valodas

CV parsēšanu var veikt no biežāk izmantotajiem dokumentu formātiem (piemēram, DOC, DOCX, PDF, RTF, HTML, TIFF, TXT, XML un EML) un dažādās valodās:

-   Holandiešu
-   Angļu valoda
-   Vācu
-   Franču
-   Spāņu
-   Zviedru
-   Dāņu
-   Poļu
-   Rumāņu
-   Itāliešu
-   Slovāku
-   Čehu
-   Krievu
-   Portugāļu
-   Ķīniešu
-   Ungāru
